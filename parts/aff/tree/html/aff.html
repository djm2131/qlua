LHPC Analysis File Format (AFF)

AFF is a flexible framework for storing heterogenious data produced by LHPC designed for uniform data access by hierahical keys and exceptional scaling to large data sets. The framework consists of a portable library for manipulating AFF files, a command line utility for common operations on AFF files, and a set of converters from the existing LHPC data formats into AFF.

The library is written in C99 and compiles on both 32- and 64-bit implementations of the Standard. If you are running on a recent Linux installation you should be able to use the library.

The application code does not need to convert between big- and little-endian machines. AFF stores data in a portable binary format and transparently handles conversion of data to the target format.

The library handles large files quite well and shows linear scaling on systems that properly support files larger than 4GiB.

AFF is known to work on Linux (x86, x86-64, ppc32, ppc64) and Mac OS X (10.4 ppc).
